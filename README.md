# SNA
Empirical Analysis of Social Network Data with a focus on comparing six community detection algorithms based on data provided by the news website [derStandard.at](https://www.derstandard.at/):
* Fastgreedy
* Label Propagation
* Leading Eigenvector
* Infomap
* Multilevel
* Walktrap
